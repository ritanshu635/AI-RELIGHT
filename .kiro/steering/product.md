# Product Overview

IC-Light is an AI-powered image relighting application that manipulates illumination in images. The name stands for "Imposing Consistent Light."

## Core Functionality

- **Text-Conditioned Relighting**: Relight images based on text prompts describing desired lighting conditions
- **Background-Conditioned Relighting**: Harmonize foreground subjects with custom background images
- **AI-Powered Recommendations**: GPT-4o-mini provides contextual lighting suggestions based on image analysis
- **Automatic Upscaling**: All outputs are upscaled to minimum 1080p using Real-ESRGAN
- **Interactive Light Control**: Visual pointer interface for intuitive light direction selection

## Key Features

1. Modern dark-themed UI with gradient backgrounds
2. Automatic background removal using BRIA RMBG 1.4
3. Multiple lighting presets (Left, Right, Top, Bottom, None)
4. Support for various samplers (DDIM, Euler A, DPM++ 2M SDE Karras)
5. Highres fix for quality enhancement
6. Normal map estimation from consistent relighting

## Use Cases

- Professional photo editing and retouching
- Product photography lighting adjustment
- Portrait relighting for different moods/atmospheres
- Background replacement with lighting harmonization
- Creative lighting experimentation

## Commercial Considerations

The original BRIA RMBG 1.4 model is for non-commercial use. For commercial projects, replace with alternatives like BiRefNet.
